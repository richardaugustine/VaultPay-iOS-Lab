
# VaultPay – Vulnerable iOS App

VaultPay is an intentionally vulnerable iOS application created for learning
mobile application security and penetration testing.

## Vulnerabilities Included
- Insecure App Transport Security
- Client-side jailbreak detection
- Insecure authentication using UserDefaults
- Broken SSL pinning
- Biometric authentication bypass
- Keychain misuse
- Insecure file sharing
- Excessive logging

⚠️ This app is intentionally insecure. Do not use in production.
